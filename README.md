# IoT_blinds
This project allows to open and close blinds using Google Home.

Things you will need:
- ESP12E MCU with HW-588A modules
- Case for 4 AA batteries
- 4 AA batteries
- DC motor, 3.3-5V
- electric wires

Tools:
- Phillips screwdriver 

Installation:
- Install Arduino IDE
- Adjust IDE for ESP8266
- Install Blynk app
